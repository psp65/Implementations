DoublyLinkedList extends SinglyLinkedList
Driver operations:
1 --> print the next element of the list if available else throws exception
2 --> print the previous element of the list if available else throws exception
3 --> add a new node after the current node. Cursor will move to the newly created node.
4 --> removes the cursor node and move cursor to the Previous node
5 --> print list.
6 --> print list in reverse order.

Compiled on Java8.
